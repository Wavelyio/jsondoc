# Install

Add JitPack as a Maven repository:
```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

Add the dependency as normal:

```
	<dependency>
	    <groupId>com.github.Wavelyio.jsondoc</groupId>
	    <artifactId>jsondoc-core</artifactId>
	    <version>2.0.0</version>
	</dependency>
```
